# Screenshots

Add screenshots of the app here once you run it locally.

Suggested screenshots:
- `home.png` - the main page when you first open the app
- `genuine_example.png` - prediction on a normal looking review
- `suspicious_example.png` - prediction on a hype-y / fake looking review

To take a screenshot on Mac: `Cmd + Shift + 4` and select the area.
On Windows: use the Snipping Tool.

Then link them in the main `README.md` like this:

```markdown
![Home](screenshots/home.png)
![Suspicious example](screenshots/suspicious_example.png)
```
